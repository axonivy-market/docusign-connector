# DocuSign Konnektor

Der [DocuSign](https://www.docusign.com/products/electronic-signature)-Konnektor von Axon Ivy integriert elektronische Signaturen in deine Prozessanwendung. DocuSign eSignature ermöglicht die schnelle und einfache Unterzeichnung von Dokumenten und die Integration in bestehende Systeme. 

Dieser Konnektor:

- ermöglicht es Low-Code-Entwicklern, elektronische Signaturen in Axon Ivy Prozesse zu integrieren.
- basiert auf REST-Webservice-Technologien.
- bietet Zugriff auf die wesentlichen Funktionen von DocuSign eSignature

### Hauptmerkmale

- Nahtlose DocuSign-Integration: Versenden von Umschlägen, Anfordern von Signaturen und Abrufen signierter Dokumente direkt aus Axon Ivy.
- Low-Code-Hilfs-APIs: Dienst- und Hilfsklassen sowie aufrufbare Unterprozesse zum Erstellen von Umschlägen, Empfängeransichten und Lesen von Dokumenten.
- Eingebettetes und Remote-Signing: Unterstützung für iFrame-Einbettung und DocuSign JS sowie Redirect-Callback-Verarbeitung.
- Flexible Authentifizierung: Unterstützung für Authorization Code Grant und JWT-Service-Authentifizierung mit RSA-Schlüsseln.
- Case-Integration: Automatisches Anhängen signierter Dokumente an Cases und Download-Helper.
- Fertige UI-Komponente: Beinhaltet eine `DocuSignPopup`-Komponente zum Einbetten der Signatur-Oberfläche.
